# Sales-Testing-Dashboard
A simple and resposive Sales Dashboard Web Application built to track key sales matrics such as revenue, deals won and conversion rate.
This project show my journey combining Frontend Development with tech Sales concepts.

# Features 
Track total Revenue
Monitor Deals Won and View Conversion Rate
Add New Sales Records and Display Sales Data in Table Format just as a Sales Funnel
Clean and simple UI

# Problem It Solves
Many business struggle to keep track of their sales performance across different clients and deals.
This dashboard provides a simple way to:
Record sales data
Monitor performance
Make Better business decisions
